THIS PROGRAM ROTATES A BIT, BACK AND FORTH, THROUGH THE ACCUMULATOR.
SIMILAR TO THE EYES ON A CYLON ROBOT FROM BATTLESTAR GALACTICA.

THE SPEED OF THE ROTATION IS CONTROLLED BY THE SWITCH REGISTER.

TO START:
   SET THE SWITCH REGISTSER TO 0100
   PRESS LOAD ADDRESS/ADDRESS LOAD
   PRESS START (8/I) OR CLEAR AND CONTINUE (8/E)
   
A SINGLE BIT WILL ROTATE, BACK AND FORTH, THROUGH THE ACCUMULATOR
AT A RATE OF APPROXIMATELY 1 SECOND PER ROTATION.

TO SPEED UP OR SLOW DOWN THE ROTATION SPEED CHANGE THE
VALUE ON THE SWITCH REGISTER.

NOTE:  A VALUE OF 7777 ON THE SWITCH REGISTER IS ABOUT 64 SECONDS
       PER ROTATION ON A PDP-8/E.

To assemble:  palbart cylon.pa
